elfego ::: [Development> cat select.sh
select i in eeny meeny miny moe
do
echo "You chose $i"
done

elfego ::: [Development> bash select.sh
1) eeny
2) meeny
3) miny
4) moe

#? 1
You chose eeny
#? 2
You chose miny
#?

PS3 is the select prompt. When the user is given a choice in bash script this is displayed.

Copy string into your startup script .bashrc or .bash_profile. export PS3="<custom prompt string>"

Examples Click to use

  • select?
  • \[\e[0;31m\]^>?\[\e[0m\]
  • \[\e[4;35;47m\]please make selection out of the above :\[\e[0m\]
  • \[\e[1;36m\]\u\[\e[0m\] choose!:

Variables

  • \[ - to begin color code sequence
  • \] - to end color code sequence
  • \u - user
  • \w - current working directory
  • \W - current working directory (short)
  • \h - hostname
  • \d - date
  • \j - current shell jobs
  • \s - shell name
  • \t - time 24h HH:MM:SS
  • \T - time 12h HH:MM:SS
  • \@ - time 12h am\pm
  • \! - history number
  • \# - command number
  • \$ - display # if root, $ otherwise
  • \n - newline

  • \e[0;30m - black
  • \e[0;31m - red
